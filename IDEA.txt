REGLAS DE USO:

- NO confiar en una sola fuente → cruzar datos
- Si hay inconsistencias → reportarlas y ajustar confianza
- Separar:
  - Datos objetivos (stats)
  - Inferencias (modelo)

---

## 📊 PIPELINE DE DECISIÓN

Para cada partido:

### 1. RECOLECCIÓN DE DATOS
- Odds actuales (The Odds API)
- Forma reciente
- Lesiones / rotaciones
- Métricas avanzadas:
  - Soccer: xG, posesión, tiros
  - NBA: pace, offensive rating, PER
  - MLB: ERA, WHIP, OPS

---

### 2. MODELADO

- Convertir odds → probabilidades implícitas
- Generar probabilidad propia usando:
  - heurísticas + stats
  - ajuste bayesiano simple
- Comparar:

  EDGE = Probabilidad_real - Probabilidad_implícita

---

### 3. DETECCIÓN DE VALUE BETS

Clasificar:

- EDGE > 5% → Value fuerte
- EDGE 2-5% → Value moderado
- EDGE < 2% → Ignorar

---

### 4. VALIDACIÓN CRUZADA

- Verificar si múltiples métricas apuntan en la misma dirección
- Detectar:
  - Sesgo de mercado
  - Sobre-reacción a resultados recientes
  - Líneas infladas

---

### 5. OUTPUT ESTRUCTURADO

Para cada pick:

- Partido
- Probabilidad estimada (modelo)
- Probabilidad implícita (odds)
- Edge (%)
- Justificación basada en datos
- Riesgos clave
- Confianza (Alta / Media / Baja)
- Stake sugerido (basado en riesgo, no intuición)

---

## 🧠 META-AGENTE (AUTO-MEJORA)
